## Timer convention

- When Doug says **“start coding timer”**, immediately record the start time in
  this file and set the timer status to **running**.
- When Doug says **“stop coding timer”**, calculate the elapsed session time,
  append it to the session log, add it to the current total, and set the timer
  status to **stopped**.
- Keep elapsed time to the nearest second. Do not count time before the start
  command or after the stop command.
